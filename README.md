# 🧠 Mental Health Platform

A modern and responsive **Mental Health Frontend Platform** designed to support emotional well-being through a clean, calming, and user-friendly experience.

This project focuses on creating an intuitive interface for **mood tracking, journaling, self-care, and mental wellness resources**.

---

## 🌟 Overview

The goal of this project is to provide users with a safe and calming digital environment where they can:

- Track their mood daily
- Write personal journal entries
- Access wellness resources
- Stay motivated through reminders
- Experience a smooth and responsive UI

---

## ✨ Features

### 🏠 Landing Page
- Modern hero section
- Clean and calming UI
- Responsive navigation bar

### 📊 Mood Tracker
- Select daily emotions
- Mood history UI
- Visual tracking interface

### 📝 Daily Journal
- Write daily reflections
- Organized journal layout
- Minimal and distraction-free design

### 💬 Support & Resources
- Mental wellness resources page
- Self-care recommendations
- Helpful support information

### 🔔 Reminder UI
- Reminder cards
- Notification component design
- Daily wellness prompts (UI only)

### 📱 Responsive Design
- Mobile-friendly layout
- Tablet & desktop optimization
- Smooth user experience across devices
